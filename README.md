# Art Philip Studio

Sitio web del estudio con páginas para Gym, Restaurante y Barbería.

## Estructura

- `index.html` — landing principal
- `estudio-art-philip/` — páginas del estudio
  - `gym.html`
  - `restaurante.html`
  - `barberia.html`
  - `assets/` — imágenes (logo, etc.)

## Desarrollo local

Abrí `index.html` en tu navegador o usá una extensión tipo Live Server.

## Deploy

Podés publicarlo con GitHub Pages (branch `main`, carpeta `/` o `/docs`).
